# Supported codecs and workflows

**Ingest Support**

| Field                             | Notes                                                                                 |
| --------------------------------- | ------------------------------------------------------------------------------------- |
| **Location**                      | EU, US                                                                                |
| **Ingest Format**                 | RTMP                                                                                  |
| **Ingest Codec**                  | H264 + AAC                                                                            |
| **Ingest Authentication**         | Webhook based                                                                         |
| **Encoder Software**              | **Supported tested**: OBS 25.0.8, ffmpeg 4.3, Streamaxia                              |
|                                   | **Supported untested**: Xsplit, Other versions of OBS and ffmpeg, other RTMP encoders |
|                                   | **Unsupported**: WebRTC and SRT encoders                                              |
| **Resolution**                    | **Supported tested**: custom resolution up to 1080p                                   |
|                                   | **Supported untested**: custom resolution up to 4k                                    |
| **Framerate**                     | between 5fps and 60fps, passthrough or custom                                         |
| **GOP Size (Keyframe Interval) ** | **Supported tested**: Constant GOP, Variable GOP of 0.5s to 8s                        |
|                                   | **Supported untested**: Variable GOP not within 0.5s to 8s                            |

**Transcoding Support**

| Field                         | Notes                                                                   |
| ----------------------------- | ----------------------------------------------------------------------- |
| **Location**                  | EU, US                                                                  |
| **Codec**                     | H264 + AAC                                                              |
| **H264 Transcoding Profiles** | Baseline, Main, High                                                    |
| **FPS**                       | **Supported tested**: 24, 25, 30, 60 fps, uniform across all renditions |
|                               | **Supported untested**: custom fps, uniform across all renditions       |
| **Resolution**                | **Supported tested**: custom resolution up to 1080p                     |
|                               | **Supported untested**: custom resolution up to 4k                      |
| **Pixel Format**              | YUV4:2:0                                                                |
